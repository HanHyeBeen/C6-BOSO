## 📱 소개

> 청각장애인이 맥북으로 재생되는 영상의 소리를 실시간 자막과 시각적 효과로 확인하며, 몰입감 있게 시청할 수 있도록 돕는 앱

[🔗 앱스토어/웹 링크](https://example.com)

- ✅ 실시간 자막 제공 _ 맥북에서 재생되는 영상의 소리를 캡처하여 STT로 변환하여 자막을 실시간으로 표시합니다.
